# Incident Response

1. **Detection** — Sentry alerts, Prometheus error-rate spikes, `/health` degradation
2. **Triage** — On-call classifies severity (P1–P3)
3. **Containment** — Disable affected API keys, rotate secrets if compromised
4. **Notification** — Enterprise customers notified within SLA (4h P1)
5. **Post-mortem** — Document root cause and preventive controls
